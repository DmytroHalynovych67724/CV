# CV — Dmytro Halynovych

Statyczna, responsywna strona CV przygotowana do publikacji przez GitHub Pages.

## Zdjęcie

Po dodaniu własnego zdjęcia jako `photo.jpg`, w pliku `styles.css` można zastąpić tło klasy `.portrait` przez:

```css
background: url("photo.jpg") center top / cover no-repeat;
```

Następnie usuń elementy `.portrait-mark` i napis `miejsce na zdjęcie` z `index.html`.
